# Hand Meme Detector

⋆ ˚｡⋆୨♡୧⋆ ˚｡⋆

Proyecto de visión por computadora que detecta gestos de la mano
en tiempo real y muestra memes según el gesto detectado.

୨୧ ───── ୨୧

## Tecnologías usadas

⋆｡°✩ ୨୧ ✩°｡⋆

* Python 3.12
* OpenCV
* MediaPipe
* NumPy

୨୧ ───── ୨୧

## Gestos detectados

˗ˏˋ ♡ ˎˊ˗

* Pulgar arriba
* Signo de paz
* Mano abierta
* 
୨୧ ───── ୨୧

## Cómo correrlo

⋆ ˚｡⋆୨♡୧⋆ ˚｡⋆

Instala las dependencias:

```
pip install opencv-python mediapipe numpy
```

Corre el programa:

```
python main.py
```

~ ~ ~ ୨୧ ~ ~ ~

La primera vez descarga el modelo automáticamente.
Presiona Q para salir.
